# PET-Проект Vitochka

## Описание проекта

Проект представляет собой веб-приложение на базе Django, которое позволяет:
- [x] создавать и просматривать фотоальбомы
- [x] загружать и хранить фотографии
- [x] назначать права доступа к альбомам для разных пользователей
- [x] управлять контентом и правами доступа с помощью административной панели
- [x] использовать функционал суперпользователя и администратора для управления системой

## Основные функции проекта:
1. Для пользователя (заказчика):
   * просмотр доступных альбомов и фотографий
   * скачивание отдельных фотографий или альбома целиком
2. Для администратора (владельца альбома \ альбомов)
   * создание альбомов
   * выбор картинки\фотографии, которая будет обложкой альбома в списке альбомов
   * изменение порядка фотографий в альбоме
   * контроль прав доступа к альбому
   * добавление и удаление фотографий
3. Для суперпользователя
   * полный доступ к управлению проектом
   * доступ ко всем альбомам
   * управление администраторами и другими польщователями

## Установка и запуск проекта
1 клонировать репозиторий

    git clone https://github.com/Andreyarkin/vitochka.git
2 перейти в директорию проекта

    cd путь_к_проекту    
3 активировать виртуальное окружение


для Windows: 

    ll_env\Scripts\activate 
для MacOS:
      
    source ll_env/bin/activate     
4 установить зависимости:

    pip install -r requirements.txt
5 выполнить миграции:
   
    python manage.py migrate
6 создать суперпользователя:
    
    python manage.py createsuperuser
7 запустить сервер разработки:
    
    python manage.py runserver
8 открыть в браузере: <http://127.0.0.1:8000/>

## Структура проекта:
* vitochka/ - файлы управления проектом
* viapp/ - приложение для управления альбомами и фотографиями
* users/ - приложение для управления пользователями
* static/ и media/ — для хранения статических файлов и загруженных медиа
* templates/ -  шаблоны для отображения страниц
